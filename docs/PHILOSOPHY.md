# Ansinews is the response to intrusive ads that made reading news on the web impossible.

The core philosophy of Ansinews is based on the following principles:

1. Minimalism in design, implementation, and functionality: Function over form.
2. Focus on usability and ease of use: Human centered design.
3. Total control over the content: User defined news experience.
4. Safety and privacy of users: Simple code, zero dependencies, anonymous usage.

Product direction:

- one shared core for feeds, parsing, state, and commands
- one flagship terminal experience
- one lightweight browser companion that reuses the core
- features parity with optimized UI and behavior for each platform
